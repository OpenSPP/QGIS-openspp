about=OpenSPP GIS plugin enables QGIS users to:
    - View pre-computed GIS layers from OpenSPP (choropleths, indicators)
    - Query registrant statistics for selected polygon areas
    - Save geofences (areas of interest) to OpenSPP
    - Export layers for offline use

    This is a thin client plugin - all computation happens in OpenSPP/PostGIS.
    The plugin displays pre-aggregated data for performance and privacy.
